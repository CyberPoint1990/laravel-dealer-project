# Laravel Dealer Management System

A Dealer Management System built with Laravel for managing dealers, certificates, and verification.

## 🚀 Features

* Dealer Registration
* Dealer List Management
* Dealer Certificate Generation
* Public Certificate Verification
* Admin Dashboard
* Responsive UI

## 🛠 Tech Stack

* PHP
* Laravel Framework
* MySQL Database
* Blade Template Engine
* Bootstrap / Custom CSS
* JavaScript

## ⚙️ Installation Guide

### 1️⃣ Clone Repository

```
git clone https://github.com/CyberPoint1990/laravel-dealer-project.git
```

### 2️⃣ Move to Project Folder

```
cd laravel-dealer-project
```

### 3️⃣ Install Dependencies

```
composer install
```

### 4️⃣ Environment Setup

Copy `.env.example` to `.env`

```
cp .env.example .env
```

### 5️⃣ Generate Application Key

```
php artisan key:generate
```

### 6️⃣ Configure Database

Edit `.env` file:

```
DB_DATABASE=your_database
DB_USERNAME=root
DB_PASSWORD=
```

### 7️⃣ Run Migration

```
php artisan migrate
```

### 8️⃣ Start Server

```
php artisan serve
```

Application will run on:

```
http://127.0.0.1:8000
```
